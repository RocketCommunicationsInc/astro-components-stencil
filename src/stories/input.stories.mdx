# Input

Input Fields allow users to enter freeform text. Variations on this field often provide specific data entry formats such as masked data (e.g. passwords or phone numbers), date and time, and numeric data entry.

## Guidelines

-   [Astro UXDS: Input Field](https://www.astrouxds.com/components/input-field/)

### Types

The following `<input>` types are available:

* Text inputs (type="text")
* Password inputs (type="password")
* URL inputs (type="url")
* Email inputs (type="email")
* Telephone inputs (type="tel")
* Search inputs (type="search")
* Number-only inputs (type="number")
