# Push Button

Push Buttons are a variant of the toggle button which incorporate label and action into a single user interface element. Push Buttons may provide a useful interface element where screen real estate is at a premium.

## Guidelines

- [Astro UXDS: Toggle & Push Button](http://www.astrouxds.com/ui-components/toggle)
